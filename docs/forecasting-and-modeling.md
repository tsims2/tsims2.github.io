# Forecasting & Modeling Stack

## Data Preparation

- Time-series data for customer and grid load
- Price curves and emissions factors
- Enriched with weather, calendar, and program details

## Modeling Approaches

- Neural networks (e.g., LSTM) for load and price forecasting
- Monte Carlo simulation to generate scenarios
- Linear optimization for strategy evaluation under uncertainty

## Integration

Model outputs are written back to ESP and surfaced through:

- Power BI dashboards
- Scenario comparison tools
- Report generation flows
