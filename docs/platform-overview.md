# CPower Analytics Platform — Overview

The CPower Analytics Platform is an internal product that unifies data ingestion,
storage, calculation, and reporting across markets.

## Objectives

- Provide a single, trusted data layer for enrollments, settlements, intervals, and customer payments.
- Standardize analytics logic across ISOs and programs.
- Enable automation and advanced forecasting at scale.

## High-Level Layers

1. **Source Systems** — ISO portals, utility feeds, real-time meters, Dynamics 365.
2. **ESP Database** — Normalized schema for cross-market analytics.
3. **Analytics Services** — Python microservices, Azure Functions, and APIs.
4. **Reporting & Decision Support** — Power BI models, reports, and generated artifacts.
